# 通知用サウンド（各自で配置）

このディレクトリには **個人用の OGG ファイル** を置きます。リポジトリには音声バイナリは含めていません（clone 直後は空でも動作します。音だけ鳴りません）。

## 置き方

1. 好きな素材を **Vorbis（`.ogg`）** に変換する（必要なら ffmpeg など）。
2. 次のファイル名のいずれかで **`sounds/` に保存**する。

| ファイル名 | 用途 |
|------------|------|
| `commit.ogg` | チャットに「コミット」系メッセージが付いたとき |
| `awake.ogg` | Hub 設定の Awake トグル確認音 |
| `mictest.ogg` | Hub 設定の「Sound notifications」プレビュー |
| `notify_*.ogg` | 通常のチャット通知: **都度ランダム**で1つ再生（ファイルが無ければ**無音**）。候補は `/notify-sounds` で配る（順序も都度シャッフル）。`name` 省略の `/notify-sound` も同じ集合からランダム（候補が無ければ 404）。例: `notify_default.ogg`, `notify_alt.ogg` |
| `HH-MM.ogg` | 毎日その時刻に鳴らす（例: `8-00.ogg`、`20-30.ogg`。時・分は1〜2桁＋2桁） |
| `auto.ogg` | スケジュール・その他で参照される場合がある |

名前が一致するファイルが無い場合、その機能の音だけスキップされます（エラーにはなりません）。

## ライセンス

配布物に含めない音声は、**ご自身で権利の通る素材** を使ってください。
